\doxysubsection{Detailed Description}
The result of a parsing operation. 

\textbackslash{}availability {\bfseries{This type only exists when exceptions are disabled.}} Otherwise \doxylink{classparse__result}{parse\+\_\+result} is just an alias for toml\+::table\+: \textbackslash{}cpp \doxylink{_classes_2lib_2include_2pconfigintrin_8h_aaca4765a01b6f3cd5edb899a1159f303}{if} TOML\+\_\+\+EXCEPTIONS using \doxylink{classparse__result}{parse\+\_\+result} = table; \#else class \doxylink{classparse__result}{parse\+\_\+result} \{ // ... \#endif \textbackslash{}ecpp

\textbackslash{}detail A \doxylink{classparse__result}{parse\+\_\+result} is effectively a discriminated union containing either a toml\+::table or a toml\+::parse\+\_\+error. Most member functions assume a particular one of these two states, and calling them when in the wrong state will cause errors (e.\+g. attempting to access the error object when parsing was successful). \textbackslash{}cpp toml\+::parse\+\_\+result result = toml\+::parse\+\_\+file("{}config.\+toml"{}); if (result) do\+\_\+stuff\+\_\+with\+\_\+a\+\_\+table(result); //implicitly converts to table\& else std\+::cerr \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} "{}\+Parse failed\+:\textbackslash{}n"{}sv \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} result.\+error() \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} "{}\textbackslash{}n"{}; \textbackslash{}ecpp

\textbackslash{}out example output\+:

Parse failed\+: Encountered unexpected character while parsing boolean; expected \textquotesingle{}true\textquotesingle{}, saw \textquotesingle{}trU\textquotesingle{} (error occurred at line 1, column 13 of \textquotesingle{}config.\+toml\textquotesingle{}) \textbackslash{}eout

Getting node\+\_\+views ({\ttfamily operator\mbox{[}\mbox{]}}, {\ttfamily \doxylink{classparse__result_a0eda76e47ce514a80e2a998621d2e641}{at\+\_\+path()}}) and using the iterator accessor functions ({\ttfamily \doxylink{classparse__result_a670ad11bdbaf1773aa00ae5ab19e66a0}{begin()}}, {\ttfamily \doxylink{classparse__result_a4a5a65c875a38a8248ac3b7fd3fc7669}{end()}} etc.) are unconditionally safe; when parsing fails these just return \textquotesingle{}empty\textquotesingle{} values. A ranged-\/for loop on a failed \doxylink{classparse__result}{parse\+\_\+result} is also safe since {\ttfamily \doxylink{classparse__result_a670ad11bdbaf1773aa00ae5ab19e66a0}{begin()}} and {\ttfamily \doxylink{classparse__result_a4a5a65c875a38a8248ac3b7fd3fc7669}{end()}} return the same iterator and will not lead to any dereferences and iterations.
