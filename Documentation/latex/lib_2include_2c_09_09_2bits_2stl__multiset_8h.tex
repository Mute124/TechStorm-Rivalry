Meets the requirements of a \href{tables.html\#65}{\texttt{ container}}, a \href{tables.html\#66}{\texttt{ reversible container}}, and an \href{tables.html\#69}{\texttt{ associative container}} (using equivalent keys). For a {\ttfamily multiset$<$\+Key$>$} the key\+\_\+type and value\+\_\+type are Key.

Multisets support bidirectional iterators.

The private tree data is declared exactly the same way for set and multiset; the distinction is made entirely in how the tree functions are called (\texorpdfstring{$\ast$}{*}\+\_\+unique versus \texorpdfstring{$\ast$}{*}\+\_\+equal, same as the standard).
