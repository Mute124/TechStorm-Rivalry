Detailed Description

The Listener is a singleton object used to control your position in the virtual space. With sound positions and listener position (and rotation) you control how the sound output is spread out over the available speakers.

Also the listener keeps track of its own velocity, ie. the speed at which the object is moving. This is used in doppler calculations.
