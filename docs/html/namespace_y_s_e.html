Detailed Description

This class can hold the properties of a particular audio device. You're not supposed to create object of this class yourself, but you can retrieve the available audio devices from the System() object.

TODO: Decide what we will keep: these generic functions or the filleBuffer class.

These functions operate on basic buffers. They are not part of the buffer class because it's better to keep it as small as possible.
