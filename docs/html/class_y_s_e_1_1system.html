◆ getGlobalReverb()

reverb & YSE::system::getGlobalReverb ( )

Get a reference to the global reverb object. It's not active by default, but when enabled, these reverb settings will be used when there's no other reverb active at the current position. If one or more reverbs are partially active (rolloff distance) the global reverb is added partially.
